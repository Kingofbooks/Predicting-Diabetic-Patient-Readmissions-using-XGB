# Medical Readmission Prediction using XGBoost

This project applies XGBoost to handle imbalanced medical datasets for predicting diabetic patient readmissions.

## Dataset
Diabetic Patients Readmission Dataset (Kaggle)

## Techniques Used
- XGBoost classifier
- SMOTE oversampling
- Class weighted boosting
- ROC Curve
- Precision-Recall Curve

## Results
SMOTE combined with XGBoost achieved the best performance in predicting minority class.

## Tools
Python, Pandas, Scikit-learn, XGBoost, Imbalanced-learn, Matplotlib
